#Syntax Highlighting
git clone git://github.com/zsh-users/zsh-syntax-highlighting.git ~/.oh-my-zsh/custom/plugins/zsh-syntax-highlighting

Open up ~/.zshrc and add zsh-syntax-highlighting to the end of the plugins=( ... ) array.
